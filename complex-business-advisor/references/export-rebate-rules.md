# 出口退税规则手册

## 一、出口退税基本概念

### 1.1 出口退税定义

出口退税是指对出口货物已征收的国内增值税、消费税部分或全部退还给出口企业的一种税收优惠政策。其目的是使出口货物以不含税价格进入国际市场，提高竞争力。

### 1.2 出口退税适用范围

**可以退（免）税的货物**：
- 自营出口或委托出口的货物
- 来料加工复出口的货物
- 进料加工复出口的货物
- 境外承包工程货物
- 保税区、出口加工区销售的货物

**不可以退（免）税的货物**：
- 援外出口货物（除另有规定外）
- 没有出口经营权的非生产型企业出口货物
- 国家禁止出口的货物
- 出口应税消费品（退税率为0，征多少退多少）

## 二、出口企业类型

### 2.1 生产型企业

**定义**：具有生产能力的企业，包括自营出口和委托出口。

**适用退税办法**：免抵退办法

**计算公式**：
```
当期免抵退税不得免征和抵扣税额 = 当期出口货物离岸价 × 外汇人民币折合率 ×（出口货物退税率 - 适用税率）

当期应纳税额 = 当期内销货物销项税额 -（当期进项税额 - 当期免抵退税不得免征和抵扣税额）- 上期留抵税额

当期免抵退税额 = 当期出口货物离岸价 × 外汇人民币折合率 × 出口货物退税率

当期应退税额 = min（当期应纳税额绝对值，当期免抵退税额）
```

### 2.2 外贸型企业

**定义**：不具有生产能力，收购货物出口的企业。

**适用退税办法**：先征后退办法

**计算公式**：
```
应退税额 = 收购不含税购进金额 × 退税率
```

## 三、退税率与商品分类

### 3.1 退税率档次

根据财政部、税务总局公告，出口货物退税率分为以下档次：
- 13%：适用增值税税率13%的货物
- 9%：适用增值税税率9%的货物（农产品、饲料、化肥等）
- 6%：部分货物和服务
- 0%：取消退税或不予退税的货物

### 3.2 主要商品退税率参考

| 商品类别 | 退税率 |
|---------|--------|
| 机电产品 | 13% |
| 纺织品、服装 | 13% |
| 钢材 | 0-13% |
| 化工产品 | 6-13% |
| 农产品 | 9% |
| 信息技术产品 | 13% |

**注**：具体退税率以申报时税务局确认的为准

## 四、报关单与发票匹配规则

### 4.1 匹配率要求

**最低匹配率**：≥98%

报关单与发票匹配率计算公式：
```
匹配率 = （匹配报关单金额 / 出口报关单总金额） × 100%
```

### 4.2 匹配要素

报关单与出口发票必须一致的要素：
- 出口货物名称（中英文）
- 商品规格型号
- 数量
- 成交方式（FOB/CIF等）
- 金额（允许合理损耗范围内的微小差异）

### 4.3 不匹配处理

**允许微小差异的情况**：
- 汇率四舍五入导致的差异（差异率≤0.5%）
- 商业折扣导致的合理差异

**不允许的差异**：
- 商品名称严重不符
- 金额差异超过1%
- 数量差异超过5%

## 五、汇率换算规则

### 5.1 汇率来源

使用**中国人民银行公布的申报当期基准汇率**（中间价）

### 5.2 汇率确定时点

- **月度申报**：使用每月1日的中间价
- **季度申报**：使用季度首月1日的中间价

### 5.3 汇率折算公式

```
人民币金额 = 美元金额 × 汇率中间价
```

## 六、换汇成本检查

### 6.1 换汇成本计算公式

```
换汇成本 = 采购不含税进价（元人民币）/ 出口美元离岸价（美元）
```

### 6.2 合理区间

| 区间 | 含义 |
|-----|------|
| <5 | 成本异常偏低，可能存在骗税风险 |
| 5-8 | 正常区间 |
| >8 | 成本偏高，出口竞争力下降 |

### 6.3 异常预警

以下情况触发预警：
- 换汇成本<5或>8
- 同一供应商集中供货
- 商品价格波动异常

## 七、出口退税申报流程

### 7.1 申报时间

- **生产型企业**：每月15日前（遇节假日顺延）
- **外贸型企业**：每月15日前（遇节假日顺延）

### 7.2 申报资料

**首次申报需提供**：
- 营业执照副本
- 对外贸易经营者备案登记表
- 海关进出口货物收发货人备案回执
- 出口退（免）税备案表

**日常申报需提供**：
- 报关单
- 出口发票
- 采购发票（增值税专用发票）
- 装箱单、提单
- 收汇凭证

### 7.3 申报系统

使用**出口退税离线申报系统**或**电子税务局出口退税申报模块**

---

## 八、风险防控要点

### 8.1 常见风险点

1. **单证不齐**：缺少报关单、发票等必要单证
2. **商品名称不符**：报关单与发票商品名称不一致
3. **汇率使用错误**：未使用央行中间价
4. **换汇成本异常**：超出合理区间
5. **申报超时**：超过规定申报期限

### 8.2 合规建议

1. 建立完善的单证管理制度
2. 定期核对报关单与发票匹配率
3. 关注退税率调整公告
4. 保持与税务机关的良好沟通
5. 保留完整的出口业务档案（至少10年）
