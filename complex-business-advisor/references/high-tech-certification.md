# 高新技术企业认定指南

## 一、高新技术企业认定概述

### 1.1 高新企业认定意义

- 企业所得税率由25%降至15%
- 提升企业品牌形象
- 更容易获得融资和项目支持
- 享受地方财政奖励

### 1.2 认定有效期

高新技术企业管理有效期为**三年**，到期需重新认定

### 1.3 认定时间

- 每年受理一般为5月、8月、10月（具体以当地科技部门通知为准）
- 评审周期约2-3个月

---

## 二、认定条件（6项核心指标）

### 2.1 企业成立时间

**要求**：企业申请认定时须注册成立**一年以上**

**计算方式**：
```
认定当年 - 企业注册成立年份 ≥ 1年
```

**注**：不含当年

### 2.2 高新技术产品（服务）收入占比

**要求**：近一年高新技术产品（服务）收入占企业同期总收入的比例**≥60%**

**计算公式**：
```
高新收入占比 = 高新技术产品（服务）收入 / 企业同期总收入 × 100%
```

**高新技术产品（服务）定义**：
- 对企业主要产品（服务）发挥核心支持作用的技术属于《国家重点支持的高新技术领域》规定范围的产品（服务）
- 需具备有效的知识产权所有权

### 2.3 研发费用占比

**要求**：近三年研发费用占销售收入的比例需符合以下标准：

| 销售收入 | 研发费用占比要求 |
|---------|----------------|
| 销售收入 < 5000万元 | ≥5% |
| 5000万元 ≤ 销售收入 ≤ 2亿元 | ≥4% |
| 销售收入 > 2亿元 | ≥3% |

**研发费用归集范围**：
- 人员人工费用
- 直接投入费用
- 折旧费用与长期待摊费用
- 无形资产摊销费用
- 设计费用
- 装备调试费用与试验费用
- 委托外部研究开发费用
- 其他费用

### 2.4 科技人员占比

**要求**：企业从事研发和相关技术创新活动的科技人员占企业当年职工总数的比例**≥10%**

**计算公式**：
```
科技人员占比 = 科技人员数 / 企业职工总数 × 100%
```

**科技人员定义**：
- 直接从事研发和相关技术创新活动
- 专门从事研发活动的管理
- 提供直接技术服务

**统计口径**：
- 通常用全年月平均数
- 需签订劳动合同或劳务合同

### 2.5 企业所得税税率

**要求**：高新技术企业所得税率优惠为**15%**

**条件**：企业申请认定前一年未发生重大安全、重大质量事故或严重环境违法行为

### 2.6 研发组织管理水平

**要求**：通过评分制考核，需达到相应分数线

## 三、评审指标体系（4大类100分）

### 3.1 知识产权（≤30分）

| 评分项 | 分值 | 评分标准 |
|-------|-----|---------|
| 技术的先进程度 | ≤8 | 国际领先(8)；国际先进(6)；国内领先(4)；国内先进(2)；其他(0) |
| 对主要产品（服务）在技术上发挥核心支持作用 | ≤8 | 强(8)；较强(6)；一般(4)；弱(2)；无(0) |
| 知识产权数量 | ≤8 | 1项及以上（Ⅰ类）发明专利等(8)；5项及以上（Ⅱ类）实用新型(6)；～4项(4)；1-3项(2)；0项(0) |
| 知识产权获取方式 | ≤6 | 自主研发(6)；受让受赠并购(4)；独占许可(3)；其他(0) |

### 3.2 科技成果转化能力（≤30分）

| 转化能力等级 | 分值 | 标准 |
|-------------|-----|------|
| 转化能力强 | 25-30 | 转化≥5项 |
| 转化能力较强 | 19-24 | 转化4项 |
| 转化能力一般 | 13-18 | 转化3项 |
| 转化能力较弱 | 7-12 | 转化2项 |
| 转化能力弱 | 1-6 | 转化1项 |
| 无转化 | 0 | 0项 |

**科技成果定义**：
- 专利、版权、集成电路布图设计等知识产权
- 合同技术诀窍
- 省级以上部门认定的科技成果

### 3.3 研发组织管理水平（≤20分）

| 评分项 | 分值 | 评分标准 |
|-------|-----|---------|
| 制定了企业研究开发的组织管理制度 | ≤6 | 建立了研发费用专账管理(6)；建立了辅助账(4)；未建立(0) |
| 设立了内部科学技术研究开发机构 | ≤6 | 建立了研发机构(6)；未建立(0) |
| 建立了成果转化的组织实施与激励奖励制度 | ≤4 | 建立了制度(4)；未建立(0) |
| 建立了科技人员的培养进修、职工技能培训 | ≤2 | 有培训制度(2)；无培训(0) |
| 建立了开放式创新创业平台 | ≤2 | 有平台(2)；无平台(0) |

### 3.4 企业成长性（≤20分）

| 指标 | 计算方式 | 分值 |
|-----|---------|-----|
| 净资产增长率 | （净资产总额增长额/期初净资产总额）× 100% | ≤10 |
| 销售收入增长率 | （销售收入增长额/期初销售收入）× 100% | ≤10 |

**评分标准**：
- ≥35%：9-10分
- ≥25%：7-8分
- ≥15%：5-6分
- ≥5%：3-4分
- >0%：1-2分
- ≤0%：0分

### 3.5 达标线

**认定要求**：评审总分**≥71分**

## 四、账务处理要求

### 4.1 研发费用科目设置

建议设置以下科目：
```
管理费用
├── 研发支出（也可单独设置一级科目）
│   ├── 研发支出-资本化支出
│   │   ├── 人员人工
│   │   ├── 直接投入
│   │   ├── 折旧费用
│   │   ├── 无形资产摊销
│   │   ├── 设计费用
│   │   ├── 装备调试费
│   │   ├── 委托外部研发
│   │   └── 其他费用
│   └── 研发支出-费用化支出
│       └── （同上费用类别）
```

### 4.2 辅助账设置要求

每个研发项目需建立辅助账：
- 项目名称和编号
- 项目起止时间
- 费用明细
- 凭证号
- 研发人员名单

### 4.3 高新产品收入确认

**收入确认原则**：
- 符合《企业会计准则第14号——收入》规定
- 产品具有核心技术所有权或使用权
- 收入金额能够可靠计量
- 经济利益很可能流入企业

**备查资料**：
- 高新产品销售合同
- 技术属性说明
- 知识产权证书
- 销售发票
- 收款凭证

## 五、认定申报流程

### 5.1 申报时间

一般为每年5月、8月、10月（具体以当地通知为准）

### 5.2 申报材料清单

1. 高新技术企业认定申请书
2. 企业营业执照副本复印件
3. 知识产权相关材料
4. 科研项目立项证明
5. 科技成果转化证明材料
6. 研究开发组织管理水平材料
7. 高新技术产品（服务）关键技术和技术指标说明
8. 专项审计报告（近三年研发费用、近一年高新产品收入）
9. 财务会计报告（近三年）
10. 职工和科技人员情况说明

### 5.3 认定流程

```
企业自我评价 → 注册登记 → 提交材料 → 专家评审 → 认定报备 → 公示公告 → 颁发证书
```

---

## 六、认定后管理

### 6.1 报告事项

有效期内企业发生以下重大事项，需在15日内向认定机构报告：
- 合并、分立、破产、解散
- 重组、转让、搬迁
- 主营业务变更
- 重大安全、质量事故
- 环境违法行为

### 6.2 年度发展情况报表

每年5月31日前填报《高新技术企业年度发展情况报表》

### 6.3 有效期满复审

有效期满前三个月内提出复审申请，提交近三年研发情况和高新产品收入情况

## 七、常见问题解答

### Q1：分公司能否单独申请高新认定？

**A**：不能，高新企业认定以企业法人为整体，分公司不是独立法人。

### Q2：研发费用占比如何计算三年平均值？

**A**：按近三年研发费用总额除以近三年销售收入总额计算，不是简单平均。

### Q3：购买专利能否算作自主研发？

**A**：不能算自主研发，但可作为知识产权，需注意对主要产品发挥核心支持作用。

### Q4：科技人员需要社保缴纳证明吗？

**A**：需要，需提供劳动合同和社保缴纳记录证明。

### Q5：申报前一年转让的专利还能用吗？

**A**：申报前一年内转让的知识产权，如果受让方不是本企业，则不能使用。
