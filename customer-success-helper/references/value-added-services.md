# 增值服务项目详情

## 服务分类总览

| 服务大类 | 服务子类 | 适用阶段 | 毛利率 |
|---------|---------|---------|-------|
| 基础合规 | 税务报道、变更登记 | 初创期 | 30-40% |
| 资质认定 | 高新认定、专精特新 | 成长期 | 50-60% |
| 出口业务 | 出口退税、海关备案 | 成长期 | 40-50% |
| 税收筹划 | 税务健康诊断、筹划方案 | 成熟期 | 60-70% |
| 高端咨询 | 股权架构、上市辅导 | 成熟期 | 70-80% |
| 转型服务 | 注销清算、传承规划 | 转型期 | 50-60% |

---

## 一、资质认定类

### 1.1 高新技术企业认定

**服务内容**：
- 研发费用归集与核算
- 知识产权规划指导
- 高新收入确认
- 认定材料准备
- 专家评审辅导
- 认定后维护

**适用客户**：
- 有自主研发能力
- 营收达到一定规模
- 拥有自主知识产权
- 成长期或成熟期企业

**服务周期**：6-12个月

**收费标准**：3-8万元（视难度而定）

**预期收益**：
- 企业所得税率从25%降至15%
- 研发费用加计扣除
- 地方性补贴（各地不同）

**关键节点**：
```
月份1-2  →  研发项目梳理
月份3-4  →  知识产权布局
月份5-6  →  财务数据规范
月份7-8  →  材料编制
月份9    →  专家评审
月份10   →  公示
月份11-12→  证书发放
```

---

### 1.2 专精特新企业认定

**服务内容**：
- 认定条件评估

**适用客户**：
- 中小型企业

- 成长期企业

**服务周期**：3-6个月

**收费标准**：2-5万元

---

### 1.3 一般纳税人升级

**服务内容**：
- 升级条件评估
- 申请材料准备
- 税局沟通协调
- 升级后辅导

**适用客户**：
- 年营业额接近一般纳税人标准
- 下游客户需要增值税专用发票
- 初创期向成长期过渡

**服务周期**：1-2个月

**收费标准**：5000-15000元

## 二、出口业务类

### 2.1 出口退（免）税代理

**服务内容**：
- 出口退（免）税备案
- 报关单据审核
- 退税申报操作
- 税务核查应对
- 退税进度跟踪
- 账务处理指导

**适用客户**：
- 有出口业务的外贸企业
- 跨境电商企业
- 有涉外服务的企业

**服务周期**：年度服务（每月申报）

**收费标准**：
- 小规模纳税人：1000-2000元/月
- 一般纳税人：2000-5000元/月

**预期收益**：
- 加快资金回笼
- 降低税负成本
- 专业合规保障

---

### 2.2 出口退税培训

**服务内容**：
- 出口退税政策解读
- 退税流程讲解
- 单据管理规范
- 常见问题处理
- 实操演练

**适用客户**：
- 刚开展出口业务
- 内部人员需要系统培训
- 计划自营出口的企业

**服务周期**：一次性培训（2-4小时）

**收费标准**：3000-8000元/次

### 2.3 海关备案与外汇登记

**服务内容**：

- 电子口岸卡办理
- 外汇管理局备案
- 银行外汇账户开设

**适用客户**：
- 新开展出口业务
- 需要出口退税
- 有收汇需求

**服务周期**：1-3个月

**收费标准**：5000-15000元（打包服务）

## 三、税收筹划类

### 3.1 税务健康诊断

**服务内容**：
- 税务风险全面排查
- 税负结构分析
- 优惠政策梳理
- 风险整改建议
- 健康度评分

**适用客户**：
- 成熟期企业
- 存在税负异常
- 经历税务稽查
- 准备融资上市

**服务周期**：1-2个月

**收费标准**：2-10万元

**输出成果**：
```json
{
  "health_score": 75,
  "risk_items": [
    {"category": "增值税", "risk_level": "中", "description": "进项税额抵扣不规范"},
    {"category": "企业所得税", "risk_level": "低", "description": "成本列支依据不足"}
  ],
  "tax_saving_potential": 500000,
  "recommendations": ["建议1", "建议2"]
}
```

---

### 3.2 研发费用加计扣除

**服务内容**：
- 研发项目立项指导
- 研发费用归集培训
- 费用台账建立
- 申报资料准备
- 后续跟踪服务

**适用客户**：
- 有研发投入的企业
- 制造类、科技类企业
- 高新技术企业

**服务周期**：年度服务

**收费标准**：
- 基础服务：5000-10000元/年
- 全流程服务：2-5万元/年

**预期收益**：
- 研发费用加计75%（制造业）/ 100%（科技型中小企业）
- 显著降低企业所得税

---

### 3.3 税收筹划方案

**服务内容**：
- 企业现状诊断
- 筹划空间识别
- 方案设计
- 风险评估
- 落地辅导
- 效果跟踪

**适用客户**：
- 年营收>1000万
- 税负率偏高
- 成熟期企业
- 有上市计划

**服务周期**：3-6个月（含执行跟踪1年）

**收费标准**：5-30万元（视企业规模）

**核心方法**：
```
1. 业务模式优化
   └── 拆分业务、调整定价、改变结算方式

2. 组织架构调整
   └── 分子公司设立、股权架构优化

3. 区域性优惠利用
   └── 园区税收优惠、产业扶持政策

4. 行业性优惠申请
   └── 即征即退、免税收入

5. 时间性差异安排
   └── 递延收入、提前成本
```
