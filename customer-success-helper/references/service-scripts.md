# 推荐话术模板库

## 一、开场话术

### 1.1 常规关怀开场

```
标准模板：
"您好，[客户名称]的[对接人]，我是[代账公司]的客户成功经理[姓名]。
想跟您做一个简短的回访，大概3-5分钟，了解一下近期服务体验，
您现在方便吗？"

变体（短信/微信）：
"您好[姓名]，我是[公司]小[姓]，想跟您做个服务回访，方便时回复1~"
```

### 1.2 阶段升级恭喜话术

```
模板：
"恭喜恭喜！刚看到系统提示，[客户名称]已经升级为一般纳税人了，
这是公司发展的重要里程碑！

成为一般纳税人后，您公司可以：
- 抵扣进项税额，降低采购成本
- 开具增值税专用发票，提升客户信任
- 享受更多税收优惠政策

接下来有一项重要事项需要和您同步：[具体事项]
您看什么时间方便，我给您详细说明？"

变体（针对出口企业）：
"恭喜升为一般纳税人！这样您就可以申请出口退税了，
预计每年可以帮您退回[估算金额]万元左右的税款。
我们可以安排一次免费的政策解读，您看周三还是周四方便？"
```

### 1.3 政策变动通知话术

```
模板：
"您好[姓名]，有个重要政策更新想第一时间通知您。

[政策名称]即将实施/发生变动：
- 变化内容：[简述]
- 实施时间：[时间]
- 对您公司影响：[具体说明]

建议您提前准备以下材料：[清单]
我们可以提供[具体服务]，帮助您平稳过渡。

您看需要我发一份详细的解读资料给您吗？"
```

### 1.4 流失预警关怀话术

```
橙色预警 - 主动关怀：
"您好[姓名]，最近[公司]的业务发展得怎么样？

我注意到[具体信号，如：近两个月票据提交频率有所下降]，
想了解一下是不是有什么变化：
- 业务上有没有遇到什么困难？
- 我们的服务有没有需要改进的地方？
- 有没有什么我们可以帮到您的？

您的反馈对我们非常重要，也希望能更好地支持到您的发展。"

红色预警 - 深度沟通：
"[姓名]，您好。

今天特意打电话给您，是因为我们非常重视[公司名称]的长期合作。
近期我们观察到[具体信号]，加上[外部因素，如：了解到教培行业最近政策压力较大]，
有些担心服务上是否还能完全匹配您的发展需求。

您看能否安排一个时间，我们当面沟通一下？
主要是想：
1. 了解您对公司发展的规划
2. 听听您对我们服务的意见
3. 探讨是否有更好的合作方式

您周四下午或周五上午方便吗？我可以去您公司拜访。"
```

## 二、增值服务推荐话术

### 2.1 税务筹划推荐

```
开场：
"[姓名]，跟您做个分享。

刚完成咱们公司上季度的税务健康分析，发现一个可以优化税负的机会。
如果做好的话，预计每年可以节省[估算金额]万左右的税款。

这个机会来自于：[具体原因，如：研发费用加计扣除]

想给您详细介绍一下，您看这周三下午还是周四上午方便？"

价值说明：
"简单给您算一下：

假如贵司年研发投入100万
- 不做筹划：企业所得应税 = 100万 × 25% = 25万
- 做加计扣除：应税所得 = 100万 - 100万×100% = 0，税负为0
- 节省税款：25万

而且这个是合规合法的，通过调整财务核算方式实现。
已有不少客户通过这种方式降低了税负。"

异议处理：
客户："这个靠谱吗？"
回复："完全靠谱。这是国家鼓励创新的政策红利，税法明确规定。
我们在[公司]做了多年，积累了丰富经验，已经帮助[数字]家客户享受到这个政策。

而且整个过程都是合规操作，不需要您改变业务模式。"

行动引导：
"我看您这周三下午3点有空，我来您公司做一个详细的方案说明。
带上您最近的财务报表，我们现场给您算一下具体能节省多少。
您看方便吗？"
```

### 2.2 高新认定推荐

```
开场：
"[姓名]，有个好消息想告诉您。

根据我们对贵司的评估，贵公司完全符合高新技术企业的认定条件：

[评估依据]
- 研发费用：[金额]，占比[X]%
- 知识产权：已拥有[数量]项
- 高新技术收入：[金额]，占比[X]%

如果认定成功，好处非常明显：
1. 企业所得税率从25%降至15%，税负降低40%
2. 研发费用加计扣除，还能再省一笔
3. 地方补贴[金额]
4. 品牌价值提升，有利于融资和投标

整个认定过程我们会全程服务，您只需要配合提供材料即可。"

价值量化：
"简单算一下：

贵司年应纳税所得额约[金额]
- 不认定：税 = [金额] × 25% = [税款]
- 认定后：税 = [金额] × 15% = [税款]
- 节省： [节省金额]

加上研发费用加计扣除和地方补贴，综合收益超过[总金额]

行动建议：
"建议您尽快启动，今年申报的话，明年就能享受优惠。
我们来安排一个启动会，把需要您配合的工作清单列出来，您看下周几方便？"
```

### 2.3 出口退税推荐

```
开场：
"[姓名]，恭喜！看到贵司最近开始有出口业务了。

出口业务有一个很重要的利好政策——出口退税。
简单说，就是出口的商品在报关出口后，可以申请退回之前缴纳的增值税。

这对您来说意味着什么？
- 降低产品成本，提升国际竞争力
- 加快资金回笼
- 合规保障，避免税务风险

贵司如果自己做的话，需要：
- 了解复杂的申报流程
- 准备全套单据
- 与税局、海关协调

我们可以提供全流程代理服务，让您省心省力。
您看要不要我发一份详细的服务方案给您？"

异议处理：
客户："我们自己也能做吧？"
回复："可以的，但坦白说，出口退税申报流程比较复杂，需要专业经验。
我见过不少企业自己做，经常出现单据不全、申报错误的问题，
轻则退税延迟，重则影响企业信用。

我们做这个很多年了，有专业的团队和丰富的经验，能帮您：
- 单据审核把关，确保一次通过
- 申报系统操作，省去您学习成本
- 应对税局核查，有问题我们来处理

算下来，代理费用可以从退税款中覆盖，还有净收益。"
```

### 2.4 财务规范辅导推荐

```
开场：
"[姓名]，有个话题想跟您探讨一下。

我们注意到贵司最近在[融资/投标/客户要求]方面有新的规划。
这些业务对财务规范性要求都比较高。

做一个评估：

贵司目前财务状况：
- 账目清晰度：[评估]
- 内控流程：[评估]
- 票据合规性：[评估]

如果要达到[目标]的要求，还有一些提升空间。

我们有一项[财务规范辅导]服务，可以帮您：
1. 梳理现有财务问题
2. 建立规范的账务体系
3. 完善内控流程
4. 通过专业机构审计

这样您在[融资/投标/客户审核]的时候，财务数据就是您的加分项。

要不要详细了解一下？"
```

## 三、流失挽回话术

### 3.1 初步接触

```
"您好[姓名]，我是[代账公司]的[姓名]。

今天打电话给您，主要是出于关心。
我们注意到贵司与我们合作期间，互动频率有所下降，
想了解一下是不是有什么变化。

[停顿，等待客户回应]

无论是因为业务调整，还是对我们的服务有什么建议，
我都非常希望能听到您的真实想法。

您的反馈能帮助我们做得更好。"
```

### 3.2 异议处理

```
客户："价格太贵了"
回复：
"非常理解您的考虑。

我想确认一下，您觉得贵主要是和什么相比？
[等待回应]

我们的定价是基于[服务质量/专业团队/响应速度]来定价的。
坦白说，市场上有比我们便宜的，也有比我们贵的。

如果只看价格的话，确实有一些选择。
但我想了解的是，除了价格，还有没有其他让您觉得不太满意的地方？
[挖掘真正原因]

其实我们也服务过一些最初觉得贵的客户，后来发现[价值点]之后，
反而觉得是物超所值的。"

客户："不需要那么多服务"
回复：
"完全理解。

我想确认一下，您觉得不需要的是哪些服务？
哪些服务是您觉得有价值的？

[了解客户真实需求]

明白了。其实我们可以根据您的实际需求，
量身定制一个服务方案，不一定非要打包购买。

这样吧，我约您见一面，详细聊聊您的需求，
我来做几个方案给您选择，您看怎么样？"
```

### 3.3 挽回承诺

```
"非常感谢您愿意跟我说实话。

针对您提到的问题，我已经记录下来了：
1. [问题1]
2. [问题2]

我想跟您承诺：
- [承诺1]
- [承诺2]

您能不能再给我们1-3个月的时间？
如果到时候您还是觉得不满意，我亲自帮您办理交接，确保平稳过渡。

这段时间内，我会：
- 每周跟您主动汇报工作进展
- 您有任何问题，30分钟内响应
- 每月做一次服务复盘

您觉得这样可以吗？"
```

## 四、满意度调研话术

### 4.1 调研开场

```
"您好[姓名]，我是[代账公司]的[姓名]。

占用您2-3分钟时间，做一个简单的季度服务满意度调研。
您的反馈对我们非常重要，能帮助我们持续改进服务。

现在开始，可以吗？"
```

### 4.2 调研问题

```
问题1："请您对过去一个季度的整体服务满意度打分，1-5分。"
追问："请问是几分呢？"

问题2："对服务专业度满意吗？1-5分。"

问题3："对响应速度满意吗？1-5分。"

问题4："最满意的是哪方面？"

问题5："最希望改进的是哪方面？"

问题6："您是否愿意继续与我们合作？"
- 愿意 → "非常感谢您的信任！"
- 不确定 → "请问是什么让您犹豫呢？"
- 不愿意 → "能否告诉我原因？"
```

### 4.3 调研收尾

```
"非常感谢您的宝贵时间。

针对您提到的[改进建议]，我们会认真对待并尽快改进。

如果您后续有任何问题或建议，随时联系我。
我的电话是[电话号码]。

祝您工作顺利，再见！"
```

## 六、常用话术要素

### 6.1 共情话术

```
"我非常理解您的感受。"
"您的顾虑完全合理。"
"换做是我，也会有同样的担心。"
"感谢您提出了这个问题。"
"您说得很有道理。"
```

### 6.2 提问话术

```
开放式提问：
"您能详细说说...？"
"您对这个事情怎么看？"
"您希望我们怎么改进？"

封闭式提问：
"您方便周三还是周四？"
"您是这个意思吗？"
"这样可以吗？"

确认性提问：
"我想确认一下，您说的是...？"
"您看我的理解对吗？"
```

### 6.3 承诺话术

```
"我承诺..."
"我保证..."
"我们一定..."
"请放心..."
"我亲自..."
```
