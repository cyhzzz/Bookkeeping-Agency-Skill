# 凭证生成规则

## 凭证生成原则

1. **借贷必相等**：每张凭证借贷方金额必须相等
2. **一借一贷或多借一贷**：优先采用简单分录
3. **摘要清晰**：每笔分录需有明确摘要
4. **附件完备**：必须关联原始票据
5. **日期排序**：按业务发生日期排序

---

## 票据类型与凭证模板映射

### 1. 增值税专用发票（进项）

**适用场景**：一般纳税人采购商品、接受劳务

**凭证模板**：
```
借：相关科目（不含税金额）
    应交税费-应交增值税(进项税额)（税额）
  贷：应付账款/银行存款/应付票据（价税合计）
```

**分录详解**：

| 情形 | 借方科目 | 贷方科目 |
|------|----------|----------|
| 采购办公用品 | 管理费用-办公费 | 应付账款 |
| 采购原材料 | 原材料/在途物资 | 应付账款 |
| 采购固定资产 | 固定资产 | 银行存款 |
| 接受咨询服务 | 管理费用-咨询费 | 银行存款 |
| 采购不动产 | 在建工程/固定资产 | 银行存款 |

**凭证示例**：
```
日期：2024-01-15
摘要：购入办公用品

借：管理费用-办公费         100.00
    应交税费-应交增值税(进项税额)  13.00
  贷：应付账款              113.00

附件：1张（INV-20240115001）
```

---

### 2. 增值税普通发票

**适用场景**：小规模纳税人采购、或一般纳税人采购无法取得专用发票

**凭证模板**：
```
借：相关科目（价税合计金额）
  贷：库存现金/银行存款/应付账款（价税合计金额）
```

**分录详解**：
- 税额计入成本，不单独核算进项税
- 金额为含税价

**凭证示例**：
```
日期：2024-01-16
摘要：购入办公用品（小规模纳税人）

借：管理费用-办公费         103.00
  贷：库存现金             103.00

附件：1张（INV-20240116001）
```

---

### 3. 电子发票（增值税专用/普通）

**处理规则**：
- 与纸质发票处理方式相同
- 需保存电子原件
- 在摘要中注明"电子发票"

---

### 4. 银行回单

**凭证模板**：
```
借：银行存款（收款金额）
  贷：相关科目（收款性质）
```

或

```
借：相关科目（付款金额）
  贷：银行存款（付款金额）
```

**常见类型与分录**：

| 业务类型 | 借方科目 | 贷方科目 | 说明 |
|----------|----------|----------|------|
| 收到货款 | 银行存款 | 应收账款 | 销售收款 |
| 支付货款 | 应付账款 | 银行存款 | 采购付款 |
| 提取现金 | 库存现金 | 银行存款 | 提现 |
| 支付工资 | 应付职工薪酬 | 银行存款 | 工资发放 |
| 支付手续费 | 财务费用 | 银行存款 | 银行手续费 |
| 收到利息 | 银行存款 | 财务费用-利息收入 | 存款利息 |

**凭证示例**：
```
日期：2024-01-17
摘要：收到A公司货款

借：银行存款-基本户       11300.00
  贷：应收账款-A公司     11300.00

附件：1张（银行回单BJ-20240117001）
```

---

### 5. 工资薪金发放

**凭证模板**：
```
借：应付职工薪酬-工资（应发工资合计）
  贷：银行存款/库存现金（实发工资）
      其他应付款-代扣社保公积金（个人部分合计）
      应交税费-代扣个人所得税（税款）
```

**分录详解**：
- 计提工资时：借记成本费用科目，贷记应付职工薪酬
- 发放工资时：借记应付职工薪酬，贷记上述科目

**凭证示例**：
```
日期：2024-01-20
摘要：发放2024年1月工资

借：应付职工薪酬-工资      85000.00
  贷：银行存款-基本户     74150.00
      其他应付款-社保公积金  6800.00
      其他应付款-公积金     2850.00
      应交税费-代扣个税     1200.00

附件：1张（工资表202401）
```

---

### 6. 社保公积金缴纳

**凭证模板**：
```
借：管理费用-社保费（单位部分）
    其他应付款-社保公积金（个人部分）
  贷：银行存款（合计金额）
```

**凭证示例**：
```
日期：2024-01-21
摘要：缴纳2024年1月社保公积金

借：管理费用-社保费        12800.00
    其他应付款-社保公积金   6800.00
  贷：银行存款-基本户     19600.00

附件：1张（社保缴费凭证）
```

---

### 7. 差旅费报销

**凭证模板**：
```
借：管理费用-差旅费（实际报销金额）
  贷：其他应收款-xxx（借款余额）
      库存现金/银行存款（补付/退回收支）
```

**分录详解**：
- 预借差旅费时：借记其他应收款，贷记库存现金
- 报销时：按实际支出入账，多退少补

**凭证示例**：
```
日期：2024-01-22
摘要：员工张三出差报销

借：管理费用-差旅费         3850.00
  贷：其他应收款-张三        5000.00
      库存现金              1150.00

附件：3张（机票2张、酒店1张）
```

---

### 8. 业务招待费

**凭证模板**：
```
借：管理费用-业务招待费（发生金额）
  贷：银行存款/库存现金（支付金额）
```

**限额说明**：
- 按发生额的60%扣除
- 最高不得超过当年营业收入的5‰
- 超出部分需调增企业所得税应纳税所得额

**凭证示例**：
```
日期：2024-01-23
摘要：招待客户餐费

借：管理费用-业务招待费      1500.00
  贷：银行存款              1500.00

附件：1张（餐饮发票）
```

---

### 9. 运费发票

**凭证模板**：
```
借：主营业务成本-运输费（采购成本）
    管理费用-运输费（期间费用）
  贷：银行存款/应付账款
```

**抵扣说明**：
- 9%税率的运输服务，可抵扣进项税
- 需取得增值税专用发票

**凭证示例**：
```
日期：2024-01-24
摘要：采购材料运费

借：主营业务成本-运输费      900.00
    应交税费-应交增值税(进项税额)  100.00
  贷：银行存款              1000.00

附件：1张（运费发票）
```

---

### 10. 租金发票

**凭证模板**：
```
借：管理费用-租赁费（租金金额）
    应交税费-应交增值税(进项税额)（税额，如为专用发票）
  贷：银行存款/应付账款
```

**分期支付**：
- 预付租金：借记预付账款，贷记银行存款
- 确认租金：借记管理费用，贷记预付账款

**凭证示例**：
```
日期：2024-01-25
摘要：支付2月办公室租金

借：管理费用-租赁费         10000.00
  贷：银行存款             10000.00

附件：1张（租金发票）
```

---

### 11. 水电费发票

**凭证模板**：
```
借：管理费用-水电费（实际金额）
  贷：银行存款/应付账款
```

**分账处理**（如涉及个人部分）：
```
借：管理费用-水电费（公司部分）
    应付职工薪酬-职工福利费（个人部分）
  贷：银行存款
```

---

### 12. 税金及附加

**凭证模板**：
```
借：税金及附加-城建税
    税金及附加-教育费附加
    税金及附加-地方教育附加
  贷：应交税费-应交城建税
      应交税费-应交教育费附加
      应交税费-应交地方教育附加
```

**申报缴纳时**：
```
借：应交税费-应交城建税
    应交税费-应交教育费附加
    应交税费-应交地方教育附加
  贷：银行存款
```

## 凭证生成质量检查

生成凭证后，需自动检查以下项目：

### 必检项
1. **借贷平衡**：借方合计 = 贷方合计
2. **金额正确**：每笔金额与原始票据一致
3. **科目有效**：科目代码在常用科目表中存在
4. **日期合理**：凭证日期在工作日内
5. **附件齐全**：每张凭证有关联票据

### 警告项
1. **异常科目组合**：如借方为收入类科目
2. **大额交易**：单笔金额超过50万
3. **负数金额**：需确认是否为折扣或调整
