# 票据归类规则

## 分类体系

票据按经济业务性质分为四大类：

| 类别代码 | 类别名称 | 会计科目范围 |
|----------|----------|--------------|
| A | 收入类 | 主营业务收入、其他业务收入营业外收入 |
| B | 成本类 | 主营业务成本、其他业务成本、营业外支出 |
| C | 费用类 | 管理费用、销售费用、财务费用 |
| D | 银行类 | 银行存款、库存现金、其他货币资金 |

---

## A类 - 收入类票据

### 识别特征
- 发票类型：增值税发票（销项）
- 票据来源：销售商品、提供劳务
- 典型场景：开具发票、收款收据

### 归类规则

| 情形 | 归类说明 |
|------|----------|
| 销售商品开具的增值税专用发票 | A类-主营业务收入 |
| 销售商品开具的增值税普通发票 | A类-主营业务收入 |
| 提供服务开具的服务发票 | A类-主营业务收入 |
| 出售固定资产开具的发票 | A类-营业外收入 |
| 收到的租金发票 | A类-其他业务收入 |
| 投资收益相关票据 | A类-营业外收入 |

### 异常判断
- 金额为负数 → 需人工确认（可能是退货或折扣）
- 购买方与客户名称不符 → 需人工确认
- 税率与货物名称不匹配 → 需人工确认

---

## B类 - 成本类票据

### 识别特征
- 发票类型：增值税发票（进项）
- 票据来源：采购商品、接受劳务
- 典型场景：采购发票、入库单

### 归类规则

| 情形 | 归类说明 |
|------|----------|
| 采购商品取得的增值税专用发票 | B类-主营业务成本 |
| 采购商品取得的增值税普通发票 | B类-主营业务成本 |
| 采购办公用品取得的发票 | C类-管理费用（见费用类规则） |
| 采购原材料取得的发票 | B类-主营业务成本（需结合用途判断） |
| 运费发票 | B类-主营业务成本（运输费）或C类-管理费用 |
| 采购固定资产取得的发票 | 资产入账，不归入成本类 |
| 差旅费中的住宿费发票 | C类-管理费用（见费用类规则） |

### 异常判断
- 税率异常（如：图书类应为9%但显示13%）→ 需人工确认
- 货物名称与业务不符 → 需人工确认
- 金额超过100万 → 需人工确认

---

## C类 - 费用类票据

### 费用子类划分

| 子类代码 | 子类名称 | 典型项目 | 会计科目 |
|----------|----------|----------|----------|
| C1 | 办公费 | 办公用品、文具、打印复印 | 管理费用-办公费 |
| C2 | 交通费 | 汽油费、过路费、停车费 | 管理费用-交通费 |
| C3 | 差旅费 | 机票、高铁、酒店 | 管理费用-差旅费 |
| C4 | 业务招待费 | 餐饮、娱乐 | 管理费用-业务招待费 |
| C5 | 通讯费 | 电话费、网络费 | 管理费用-通讯费 |
| C6 | 咨询服务费 | 审计费、律师费、咨询费 | 管理费用-咨询费 |
| C7 | 租赁费 | 房租、设备租赁 | 管理费用-租赁费 |
| C8 | 水电费 | 水费、电费 | 管理费用-水电费 |
| C9 | 折旧摊销 | 固定资产折旧、无形资产摊销 | 管理费用-折旧费 |
| C10 | 工资薪金 | 工资、奖金、补贴 | 应付职工薪酬 |
| C11 | 社保公积金 | 养老保险、医疗保险等 | 管理费用-社保费 |
| C12 | 职工福利费 | 福利费、体检费 | 管理费用-福利费 |
| C13 | 保险费 | 财产保险、责任保险 | 管理费用-保险费 |
| C14 | 广告宣传费 | 广告费、宣传费 | 销售费用-广告宣传费 |
| C15 | 销售费用 | 包装费、仓储费、运输费 | 销售费用 |
| C16 | 财务费用 | 利息支出、汇款手续费 | 财务费用 |
| C17 | 税金及附加 | 城建税、教育费附加、房产税 | 税金及附加 |
| C18 | 其他费用 | 无法归入上述类别的费用 | 管理费用-其他 |

### 归类规则详解

#### 办公费（C1）
- 识别关键词：办公用品、文具、纸笔、打印耗材、复印费
- 票据类型：增值税普通发票、增值税专用发票
- 税率：通常为13%或3%

#### 交通费（C2）
- 识别关键词：汽油、柴油、天然气、过路费、停车费
- 票据类型：增值税普通发票、过路费电子发票
- 特殊说明：通行费电子发票可抵扣进项税

#### 差旅费（C3）
- 识别关键词：机票、高铁票、酒店住宿
- 票据类型：航空运输电子客票行程单、铁路车票、酒店发票
- 特殊说明：机票需注明旅客信息方可抵扣

#### 业务招待费（C4）
- 识别关键词：餐饮、KTV、娱乐、旅游
- 票据类型：餐饮发票、服务发票
- 限额规定：按发生额60%扣除，但不超过营业收入5‰

#### 咨询服务费（C6）
- 识别关键词：咨询费、审计费、律师费、技术服务费
- 票据类型：增值税专用发票（6%税率）
- 特殊说明：需结合合同判断业务真实性

#### 租赁费（C7）
- 识别关键词：房租、租金、租赁费
- 票据类型：增值税专用发票或普通发票
- 税率：不动产租赁9%，动产租赁13%

#### 工资薪金（C10）
- 识别关键词：工资、薪金、奖金、补贴
- 票据类型：工资表、社保缴费凭证
- 特殊规则：通过"应付职工薪酬"科目核算，不通过发票

---

## D类 - 银行类票据

### 识别特征
- 票据来源：银行系统
- 票据形式：银行回单、银行对账单、转账记录

### 归类规则

| 情形 | 归类说明 |
|------|----------|
| 银行转账回单 | D类-银行存款 |
| 银行手续费回单 | C类-财务费用 |
| 银行利息回单 | C类-财务费用（利息收入记借方红字） |
| 贷款到账回单 | D类-银行存款（同时增加负债） |
| 还款回单 | D类-银行存款（同时减少资产/负债） |
| 支票领用 | D类-其他货币资金 |

### 银行科目细分

| 银行账户类型 | 科目代码 | 科目名称 |
|--------------|----------|----------|
| 基本户 | 1002 | 银行存款-基本户 |
| 一般户 | 1002 | 银行存款-一般户 |
| 专用账户 | 1002 | 银行存款-xxx专户 |
| 临时账户 | 1002 | 银行存款-临时户 |
| 支付宝/微信 | 1012 | 其他货币资金 |
| 信用卡 | 1002 | 银行存款-信用卡 |

## 异常预警规则

### 金额异常
```
规则编号：AMT-001
触发条件：单张发票金额 < 0
严重程度：高
处理方式：必须人工确认

规则编号：AMT-002
触发条件：单张发票金额 > 1,000,000
严重程度：中
处理方式：需人工确认业务真实性

规则编号：AMT-003
触发条件：同一供应商同一天有多张金额相近的发票
严重程度：中
处理方式：检查是否重复入账
```

### 税率异常
```
规则编号：TAX-001
触发条件：发票税率不在 [0%, 3%, 6%, 9%, 13%] 范围内
严重程度：高
处理方式：必须人工确认

规则编号：TAX-002
触发条件：货物名称与适用税率不匹配
严重程度：高
处理方式：必须人工确认
示例：货物名称为"图书"但税率为13%
```

### 重复预警
```
规则编号：DUP-001
触发条件：发票代码+发票号码已存在
严重程度：高
处理方式：禁止重复入账
```

### 票据合规性
```
规则编号：CMP-001
触发条件：发票印章不清晰或缺失
严重程度：中
处理方式：需人工确认

规则编号：CMP-002
触发条件：发票代码/号码与税务系统不一致
严重程度：高
处理方式：必须人工确认
```

## 归类优先级

当一张票据可能属于多个类别时，按以下优先级判断：

1. **收入类优先**：有明确销售属性的票据归入A类
2. **成本类次之**：有明确采购属性的票据归入B类
3. **费用类第三**：办公、差旅、招待等费用归入C类
4. **银行类最后**：银行单据类归入D类

---

## 特殊行业规则

### 小规模纳税人
- 增值税税率：3%（或减按1%）
- 进项税不得抵扣，全额计入成本
- 归类时注意区分价税合计金额

### 一般纳税人
- 增值税税率：13%、9%、6%
- 取得专用发票可抵扣进项税
- 归类时需分别记录金额和税额

### 高新技术企业
- 可享受研发费用加计扣除
- 研发费用相关票据需单独标识
- 归类时建议增加"研发费用"子类
