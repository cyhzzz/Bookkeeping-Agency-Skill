# AI代账工具

县域代理记账公司智能化工作台。基于Skill架构，支持管理层初始化定制后交付员工使用。

## 核心理念

**管理层初始化 → 品牌标识定制 → 县域逻辑微调 → 交付员工使用**

```
┌─────────────────────────────────────────────────────────────┐
│  初始化配置（管理层操作，一次性）                             │
│                                                             │
│  Step 1: 品牌信息                                          │
│  • 公司名称、简称、联系电话                                  │
│  • Logo上传 → 自动更新报告模板                              │
│                                                             │
│  Step 2: 县域实情                                          │
│  • 客户行业分布                                             │
│  • 当地税务特点                                             │
│  • 特殊优惠政策                                             │
│                                                             │
│  Step 3: 输出格式                                          │
│  • 报告内容定制                                             │
│  • 语言风格选择                                             │
│  • 文件命名规范                                            │
│                                                             │
│  Step 4: 确认节点                                          │
│  • 自动化程度选择                                           │
│  • 特殊业务复核要求                                         │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│  员工使用（日常工作，无需配置）                               │
│                                                             │
│  直接说需求：                                               │
│  • "新客户入职"                                           │
│  • "处理本月票据"                                         │
│  • "检查风险"                                             │
│                                                             │
│  AI自动识别 → 调度模块 → 生成带品牌标识的报告               │
└─────────────────────────────────────────────────────────────┘
```

## 安装

将整个文件夹作为Skill安装到AI Agent工作台：
```
D:\project\skills\代理记账\
```

## 初始化

首次使用，说"初始化"启动配置向导：

```
AI：欢迎使用AI代账工具初始化向导！
    请提供公司信息、Logo、当地实情...
```

AI自动完成：
- 品牌标识更新到所有报告模板
- 县域逻辑微调到各子Skill
- 版本号更新（v1.0.0 → v1.1.0）

## 初始化配置项

### 品牌信息
| 配置项 | 说明 |
|--------|------|
| 公司名称 | 全称，用于报告落款 |
| 简称/品牌名 | 用于文件名和简称 |
| 联系电话 | 报告联系信息 |
| Logo | 自动添加到Word报告页眉 |

### 县域实情
| 配置项 | 说明 |
|--------|------|
| 客户行业分布 | 调整行业税负率基准 |
| 税务特点 | 查账征收/核定征收 |
| 特殊优惠政策 | 更新政策追踪逻辑 |
| 申报截止习惯 | 设置提醒时间 |

### 输出格式
| 配置项 | 说明 |
|--------|------|
| 报告内容 | 月度报告包含哪些章节 |
| 语言风格 | 正式/简洁/详细 |
| 文件命名 | {客户名}_{类型}_{日期} |

### 确认节点
| 自动化程度 | 说明 |
|------------|------|
| 高自动化 | 仅关键节点确认 |
| 标准自动化 | 平衡效率与安全 |
| 严格审核 | 每步都确认 |

## 员工使用

初始化完成后，员工直接说需求即可：

| 场景 | 员工这样说 |
|------|-----------|
| 新客户入职 | "新客户入职" |
| 月度报税 | "处理本月票据" |
| 风险检查 | "检查一下这个客户的税务风险" |
| 客户价值 | "给客户出一份价值报告" |

AI自动：
1. 识别意图
2. 调度对应模块
3. 生成带品牌标识的Word报告
