# 政策变动记录

## 概述

本文档用于记录国家和省级重要税务政策变动，便于追踪和对客户进行影响分析。

## 记录格式

```json
{
  "policy_id": "POL-YYYYMMDD-XXX",
  "title": "政策标题",
  "source": "发布机构",
  "issued_date": "发布日期",
  "effective_date": "生效日期",
  "category": "政策类别",
  "summary": "变化摘要",
  "impact_level": "high|medium|low",
  "affected_customers": ["受影响客户列表"],
  "action_required": ["需要采取的行动"],
  "recorded_at": "记录时间",
  "notes": "备注"
}
```

## 2024年度重要政策变动记录

### 2024年政策

#### POL-20240101-001：小微企业所得税优惠延续

| 字段 | 内容 |
|------|------|
| **标题** | 关于小微企业和个体工商户所得税优惠政策的公告 |
| **来源** | 国家税务总局 财政部 |
| **发文日期** | 2024-01-01 |
| **生效日期** | 2024-01-01 |
| **类别** | 优惠政策 |
| **摘要** | 对小型微利企业年应纳税所得额不超过300万元的部分，减按25%计入应纳税所得额，按20%的税率缴纳企业所得税 |
| **影响程度** | 高 |
| **涉及条款** | 《企业所得税法》第二十八条 |
| **受影响客户** | 年应纳税所得额100-300万的小微企业 |
| **需采取行动** | 更新所得税预缴计算逻辑，确认客户是否仍符合小微条件 |
| **记录时间** | 2024-01-01 |
| **备注** | 优惠力度较之前有所调整，需重点关注临界点企业 |

## 数据更新要求

- **时效性**：国家政策72小时内完成记录，省级政策24小时内完成记录
- **完整性**：每条政策需包含完整的字段信息
- **可追溯性**：保留历史版本，支持政策变化追踪
- **准确性**：政策摘要需准确反映原文核心变化
